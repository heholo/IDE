Show of hands (PCA visualization)

Introduction

PCA stands for Principal Component Analysis. It is an analysis tool for reducing the dimensionality of data to a few features that capture the variance of the data. For this visualization PCA has been performed on a dataset of of 40 hand outlines.

The Visualisation

The visualisation below is in two parts. On the right, the forty hands are plotted onto the first principal component (x axis) and second principal component (y axis). The points can be selected and de-selected by clicking. For each selected point the outline of the corresponding hand is displayed on the left.

Observations

With the plot above, we can attempt to gain some intuition on what the first two principal components describe. Looking at the cluster of hands to the left (click me) we see outlines which resemble slender hands. This is in contrast to the cluster of points to the right which are thicker. This indicates that perhaps the first principal component tells us something about the width of the hand.

Looking along the second principal component instead, we see some slightly different pertubations. The top outlier displays what seems to be a broken pinky, while the bottom outlier has a fractured middle finger. Perhaps the second principal component describes finger straightness. ;)
